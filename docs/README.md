# Repository Docs

This directory is the repository-level documentation area for the parallel Python and TypeScript apps.

## Contents

- [`PROTOCOL_SPEC.md`](./PROTOCOL_SPEC.md) - shared Claude protocol reference
- [`PYTHON_SDK_ANALYSIS.md`](./PYTHON_SDK_ANALYSIS.md) - notes from the Python SDK migration work
- [`STDIO_QUICKSTART.md`](./STDIO_QUICKSTART.md) - Claude stdio protocol quick reference
- [`TODO.md`](./TODO.md) - current follow-up work
- [`plans`](./plans) - implementation and migration plans

## Scope

- Keep cross-app or repository-wide notes here.
- Keep app-specific notes close to the app under `apps/python` or `apps/typescript`.
- `apps/typescript/docs` is preserved as a historical snapshot from the restored `typescript-final` implementation.

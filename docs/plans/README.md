# Historical Plans

Most documents in this directory were written before the repository was reorganized into app subdirectories:

- `apps/python`
- `apps/typescript`

As of 2026-05-11, the historical Python app directory has been removed. These documents are kept as implementation history and may mention paths that no longer exist.

## How to read these docs

- Python paths like `src/...`, `tests/...`, `scripts/...`, `pyproject.toml`, `README.md`, and `INSTALL.md` refer to the removed historical implementation.
- TypeScript paths like `src/...`, `tests/...`, `package.json`, and TypeScript-specific docs now live under `apps/typescript/`.
- TypeScript historical design notes are preserved under `docs/plans/`.

## Intent

These files are kept as implementation history. They are not the source of truth for today's repository layout.
